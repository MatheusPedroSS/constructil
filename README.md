#Constructil

No software pretendemos utilizar algumas metodologias do SCRUM como a divisão do projeto em 3 Sprint.

1. Modulo de controle de loja (um mini ERP);
2. Modulo de orçamentos;
3. Modulo de E-commerce;

##Requisitos 1º Modulo:

###Cadastros:

- [ ] Cadastro Fornecedores;
    - [ ] CRUD.

- [ ] Cadastro Produtos.
    - [ ] CRUD.

- [ ] Cadastro Cliente.
    - [ ] CRUD.

###Estoque:

- [ ] Entrada produtos.
- [ ] Pedidos de compras.

###PDV:

- [ ] Pedidos de venda.
- [ ] Devolução de vendas.

###Relatorios:

- [ ] Financeiro;
    - [ ] Contas a pagar.
    - [ ] Contas a receber.

- [ ] Estoque;
    - [ ] Saldo produtos.
